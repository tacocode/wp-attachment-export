== Description ==

Exports only posts of type 'attachment', i.e. your media library

== Installation ==

1. Upload the folder 'wp-attachment-export' to the '/wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Export your attachments via the 'Tools > Attachment Export' menu in WordPress
